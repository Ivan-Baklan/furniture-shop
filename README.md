

## Furniture shop page

This is a [Next.js](https://nextjs.org/) furniture-shop website was built as graduation project. It shows all benefits of creating websites using Next.js framework

### Technologies stack
* Next.js
* React
* SASS
* axios
* styled-components
* Redux
* eslint(linter)
* prettier(code formating)
